# Exercises Domain

## Назначение

Домен `exercises` - центральный домен приложения. Он хранит задания, отдает публичное представление вопроса и проверяет ответы пользователя.

## Файлы

- `src/exercises/models.py`
- `src/exercises/schemas.py`
- `src/exercises/repository.py`
- `src/exercises/service.py`
- `src/exercises/router.py`

## Поддерживаемые типы упражнений

Сейчас реализованы три типа:

- `grammar`
- `translate`
- `dictation`

Тип хранится в колонке `type`, а детали упражнения лежат в JSONB-колонке `payload`.

## Связь с media

Для `dictation` упражнений поле `payload.media_filename` должно ссылаться на существующую запись в домене `media`. Если такого файла нет, сервис возвращает `400 Unknown media filename`.

## Ограничения и текущее состояние

- тип упражнения нельзя менять через `PATCH`;
- `payload` нельзя отправлять без `type` в typed-обновлении;
- список упражнений предназначен для staff, а не для публичного каталога;
- сейчас нет сущностей теста, попытки, режима экзамена, режима тренировки, модерации и тематических подборок.
