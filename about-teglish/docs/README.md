# Teglish Docs

Этот каталог фиксирует текущее состояние backend-проекта `teglish` и продуктовый контекст, который был дан отдельно.

## Список файлов

- `teglish.md` - продуктовое описание и целевое видение.
- `project-overview.md` - общий обзор проекта, архитектуры и структуры репозитория.
- `auth.md` - домен аутентификации и авторизации.
- `users.md` - домен пользователей.
- `exercises.md` - домен упражнений.
- `media.md` - домен медиафайлов для диктантов.
- `db.md` - слой базы данных, миграции и инфраструктура данных.

## Важное замечание

Сейчас медиа хранится локально через `LocalMediaStorage` в папке `uploads`, но абстракция `MediaStorage` уже выделена. Это хороший кандидат на будущую замену локального хранения на S3-совместимое хранилище.
