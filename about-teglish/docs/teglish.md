# Teglish Product Context

## Отличие приложения

- гибкость в настройке сложности прохождения квиза;
- поддержка режима экзамена и режима обучения;
- возможность добавлять свои задания;
- задание рассматривается как элемент диктанта или перевода.

## Упражнения

### Диктант по аудио/видео-отрывку

Вид упражнения можно переключать:

- составить фразу из готовых фрагментов;
- написать фразу самостоятельно.

Источники:

- аудиоотрывок из фильма, сериала, песни;
- видеоотрывок из фильма, сериала, клипа.

### Перевод

Вид упражнения можно переключать:

- несколько вариантов на выбор;
- текстовый ответ.

### Грамматика

- пропуск слова в предложении;
- варианты ответов на выбор.

## Режим прохождения квиза

- тренировочный: ошибки показываются сразу;
- экзамен: ошибки показываются только в конце квиза.

## Дополнительные фичи

- добавление своих заданий;
- подборка заданий по тематикам;
- рейтинг, где за задания даются баллы.

## Источники данных

Все возможные виды заданий пользователь может создавать сам.

### Видеотрывок

Фильмы и сериалы можно нарезать с помощью субтитров.

### Аудиоотрывок

Если аудио получено из фильма или сериала, подход аналогичен видео. Если из песни, требуется дополнительная проработка процесса.

### Перевод

- словарь можно взять из открытого источника;
- перевод можно получать с помощью Yandex API.

## Исследование

Нужно исследовать:

- нагрузочное тестирование;
- различные сетевые архитектуры;
- балансировщики нагрузки;
- воркеры;
- дальнейшее масштабирование backend-части.

## MVP
В MVP не входят:
- тематики для заданий
- режим прохождения квиза (тренировочный/экзамен)
- система рейтинга
